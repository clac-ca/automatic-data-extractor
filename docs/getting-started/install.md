# Install (local development)

Use this when developing outside the devcontainer.

## Requirements

- Python 3.14+
- Node.js 22+
- Git
- `uv` (https://astral.sh/uv)

## Setup

From the repo root:

```bash
./setup.sh
```

This installs backend dependencies into `backend/.venv/` and frontend dependencies
in `frontend/ade-web/`.
